如何将简单的网页上传到到GitHub，生成可分享任何人可点击浏览的网页链接（web方法）

  1. 将网站所需的文档准备好，包括HTML，CSS，JS，IMGS,etc.

    HTML需要命名为index.html,否则会出现404

  2. 注册GitHub，在电脑上安装好Github Desktop

  3. 在GitHub网页端创建新的repository-命名-选public／private-勾选add a README file-create

  4. 添加文件到repository：Add files-Upload files

  5. 把相关文件drag到这里：

  6. 在网页端点击设置，化到最下面设置GitHub Pages，branch选择main，然后Save
  7. 完成！
